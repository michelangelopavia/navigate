## Adattamenti mobile rispetto a desktop
- Layout single-column, tutto impilato verticalmente
- Font ridotti (titolo hero 40px vs 77px desktop)
- Padding/gap compattati (16–20px vs 48px)
- Icone card ridotte a 42×42px (da 56×56px)
- Card eventi/luoghi passano da grid a colonna singola
- Frame di riferimento: iPhone (390×844)

## Palette e stile
Identici alla versione desktop: sfondo celeste pastello, bordi/badge arancione `oklch(62% 0.19 45)`, testi/bottoni neri, font Space Grotesk.
